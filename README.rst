LTCC (Lightweight Terminal Control Center)
==========================================

Goal
==========================================

* provide convient tools like gnome-control-center (or something)
* use terminal as interface (``ncurses``)
* lightweight
* use Python 3 as main develop language
* support both X and Wayland
